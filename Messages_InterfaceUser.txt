SQS (Simple Queue Service)
  - SQS allows you to send, store, and receive messages between software components at any volume
  
SNS (Simple Notification Service)
  - Amazon SNS is similar in that it is used to send out messages to services, but it can also send out notifications to end users.
  
AWS Polly:
- Amazon Polly is a service that turns text into lifelike speech, allowing you to create applications that talk

AWS MQ:
  - is a managed message broker service for Apache ActiveMQ and RabbitMQ that makes it easy to set up and operate message brokers on AWS
  - can switch from any standards-based message broker to Amazon MQ without rewriting the messaging code in your applications
  
AWS Kinesis:
  - enables you to build custom applications that process or analyze streaming data for specialized needs
  - clickstreams, application logs, and social media to an Amazon Kinesis

AWS Kendra:
  - is an intelligent search service powered by machine learning
  - search for your websites and applications so your employees and customers can easily find the content they are looking for
  
AWS Comprehend:
  - is a natural-language processing (NLP) service that uses machine learning to uncover information in unstructured data
  - instead of combing through documents, the process is simplified and unseen information is easier to understand
